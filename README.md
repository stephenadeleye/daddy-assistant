## Setup
1. Clone repo
2. Create `.env` from `.env.example`
3. Add your `OPENAI_API_KEY` to `.env` (a placeholder is provided in `.env.example`)
4. Install dependencies: `pip install -r requirements.txt`
5. Run: `python main.py`
